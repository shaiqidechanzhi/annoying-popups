# 🎭 AdTrick Playground – 浏览器扩展开发学习示例

> 一个由 **DeepSeek V4 Pro** 辅助生成的浏览器扩展项目，用于**学习 Manifest V3 扩展开发**与**前端交互逻辑演示**。  
> 项目模拟“自动弹窗与视频广告”行为，**每个弹窗均设有明确的关闭按钮**，可随时退出，无隐藏行为。

## ✨ 设计目的（非商用 · 纯学习）

本项目并非真正的广告插件，而是一个**可控的教学演示环境**，用于展示：

- 浏览器扩展的基础架构（`manifest.json`、Service Worker、Content Scripts）
- 动态 DOM 元素注入与样式控制
- 定时器与随机事件模拟
- 用户关闭交互的响应式设计

## 🧩 当前特性一览（部分属于预期内的“演示特征”）

| 特性 | 说明 |
| :--- | :--- |
| 🎲 随机弹窗位置 | 弹窗可能出现在屏幕四个角落、中央或边缘错位，模拟不同布局 |
| 🎬 自动播放视频 | 部分弹窗内嵌短视频（静音/非静音），可正常关闭 |
| 🖱️ 始终有效的关闭按钮 | 每个弹窗都有关闭逻辑，**无任何递归弹窗或死循环** |
| 🐢 模拟“卡顿感” | 弹窗出现/关闭时加入轻微延迟或抖动，增强演示效果 |
| 📐 错位现象 | 特定分辨率下弹窗可能出现尺寸或位置偏移，属于演示的一部分 |

> 以上“特征”均为教学而设计，并非程序缺陷。

## 🧪 安装与运行（开发者模式）

1. 下载或 `git clone` 本仓库  
2. 打开 Chrome/Edge/Brave 等浏览器，进入扩展管理页面  
   - Chrome：`chrome://extensions`  
3. 开启右上角 **“开发者模式”**  
4. 点击 **“加载已解压的扩展程序”**，选择本项目的文件夹  
5. 确认加载后，插件即生效。点击浏览器工具栏中的插件图标可**启用/关闭**演示行为（如有弹出窗口控制）
